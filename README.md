# My First Bot

Инструкция по запуску бота на разных устройствах.

## 🇷🇺 Русский

Запустить файл можно на двух типах устройств: ПК и мобильный телефон.

### Вариант для телефона:
1. Скачайте приложение **Pydroid 3** из Google Play.
2. Откройте в нём файл `mini-bot.py` и запустите.

### Вариант для ПК:
1. Скачайте архив Python zip с официального сайта.
2. Распакуйте zip-архив в удобную папку.
3. Откройте файл `mini-bot.py` через установленный Python и запустите.

---

## 🇬🇧 English

You can run this file on two types of devices: PC and mobile phone.

### Phone:
1. Download the **Pydroid 3** app from Google Play.
2. Open the file and run it.

### PC:
1. Download the Python zip file from the internet.
2. Unzip the zip file.
3. Select this file and run it.
